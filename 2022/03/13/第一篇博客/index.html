# 此过程为在电脑中安装完成 nvm、npm、node、git 以及 hexo 之后，使用 hexo 向 GitHub 中上传的过程。

# 1、建立 Github 用户

# 2、创建 SSH 密钥：

# 1、运行以下命令：

# ssh-keygen -t rsa -C “建立 GitHub 账户的 邮箱” // 一直回车

# 2、在相应的文件查找生成的密钥，并全部复制

# // 一般为 C:\Users.ssh\id_rsa.pub

# 3、在 GitHub 个人页面点击头像，选择 setting 中的 SSH and GPG keys，然后点击 New SSH key，并粘贴上一步的 key

# 4、验证是否建立连接

# 运行命令：
# ssh -T git@github.com

# 3、建立 Pages 仓库，仓库名为 Github 用户名.github.io

# 4、本地在 hexo 中建立新的 md 文件并进行编辑，运行 hexo g 和 hexo s 在本地访问相应端口成功显示网页后即可进行下一 步

# 5、将 hexo 部署到 Github 上，运行 hexo d 命令，然后通过浏览 器打开 “https://Github 用户名.github,io” 即可成功

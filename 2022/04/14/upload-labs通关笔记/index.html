# 第一关：js 验证

通过 burp 进行抓包，将提前写好的 php 文件后缀改成.pg 文件后上传，将 Content-Disposition: 中的 filename 中的 jpg 后缀改回.php

# 第二关：mime-type 验证

通过 burp 进行抓包，将提前写好的 php 文件后缀改成 jpg 文件后上传，将 Content-Disposition: 中的 filename 中的 jpg 后缀改回 php，并将 Content-Type: 后改为 image/jpeg

# 第三关：黑名单验证

同第一关，但不可以使用.php 后缀，可以使用 php5，php1，phtml 等进行

# 第四关：黑名单验证

同第三关，但禁掉的后缀更多，可以使用.htaccess

# 第五关：大小写绕过

同第一关，但是更改后缀时需要大写成.PHP

# 第六关：

# 第七关：

通过 burp 进行抓包，将提前写好的 php 文件后缀改成.pg 文件后上传，将 Content-Disposition: 中的 filename 中的 jpg 后缀改回.php，并在后面加上 “.”。

# 第八关：

通过 burp 进行抓包，将提前写好的 php 文件后缀改成.pg 文件后上传，将 Content-Disposition: 中的 filename 中的 jpg 后缀改回.php，并在后面加上::$DATA。

# 第九关：

通过 burp 进行抓包，将提前写好的 php 文件后缀改成.pg 文件后上传，将 Content-Disposition: 中的 filename 中的 jpg 后缀改回.php，并在后面加上 “. .”。
